* Azure AI Search (Azure Cognitive Search):
Search as a Service
~ Elastic search
fully managed
Components:
    - Index: Where data is stored after processing
    - Indexer: Collects data from its source and ingest it to the index
    - Skillset (AI Enrichment):
        Before storing in index =>
        do some smart things like
         extracting text from images | text summarization | entity rec
    - Query API:
        Retrieves the data in the index
        Filters | Sorts
AI Search is Azure OpenAI + Vector embeddings

* OpenSearch
هو محرك بحث وتحليل بيانات مفتوح المصدر.
هو fork من Elasticsearch بعد تغيير رخصة الأخير.
يمكنه تخزين مستندات JSON بسهولة.
يدعم البحث النصي الكامل والمتقدم.
يمكنه إجراء تحليلات وaggregations في الوقت الفعلي.
يستخدم عادة لتحليل البيانات، السجلات، والـdashboards.
يمكن تشغيله على عدة nodes ويعمل clustering لتوسيع السعة.
APIs الخاصة به مشابهة لـElasticsearch، مما يسهل التحويل.
يمكن استخدامه كـdocument store أو محرك بحث أو نظام تحليلي.
مجاني ومفتوح المصدر، مع دعم تجاري عبر AWS OpenSearch Service.
