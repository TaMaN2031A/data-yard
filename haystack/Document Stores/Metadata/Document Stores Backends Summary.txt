* Azure AI Search (Azure Cognitive Search):
Search as a Service
~ Elastic search
fully managed
Components:
    - Index: Where data is stored after processing
    - Indexer: Collects data from its source and ingest it to the index
    - Skillset (AI Enrichment):
        Before storing in index =>
        do some smart things like
         extracting text from images | text summarization | entity rec
    - Query API:
        Retrieves the data in the index
        Filters | Sorts
AI Search is Azure OpenAI + Vector embeddings
